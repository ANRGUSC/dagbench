What is DAGBench?

Task graph scheduling is a foundational problem in distributed computing: given a directed acyclic graph (DAG) of dependent tasks with computation costs and communication costs between them, find an optimal mapping of tasks to heterogeneous processors that minimizes the overall completion time (makespan).

DAGBench provides a curated, diverse collection of realistic task graph workflows drawn from edge computing, IoT, scientific computing, telecommunications, and more. Every workflow ships with a standardized graph.json (task/edge costs) and metadata.yaml (provenance, domains, paper references), ready for direct use with the SAGA scheduling framework.

Unlike randomly generated DAGs, DAGBench workflows reflect real application structures—from FFT butterfly patterns and Cholesky factorizations to ML inference pipelines and UAV coordination systems.

Installation & Quick Start

CLI
# Install DAGBench $ pip install dagbench # Load a workflow and schedule with HEFT from dagbench import load_workflow from dagbench.networks import fog_network from saga.schedulers import HeftScheduler # Load any workflow by its full ID task_graph = load_workflow("edge.splitstream_pipeline") network = fog_network() # Schedule with HEFT scheduler = HeftScheduler() schedule = scheduler.schedule(network, task_graph) print(f"Makespan: {schedule.makespan:.2f}")
# Generate classic benchmark DAGs programmatically from dagbench.generators import gaussian_elimination_dag, fft_dag, cholesky_dag from dagbench.generators import mapreduce_dag, lu_decomposition_dag from dagbench.generators import random_layered_dag from dagbench.networks import fog_network from saga.schedulers import HeftScheduler # Generate a 7x7 Gaussian elimination DAG (28 tasks) task_graph = gaussian_elimination_dag(n=7) # Other classic generators fft = fft_dag(num_points=16) # 64 tasks cholesky = cholesky_dag(tiles=5) # 35 tasks lu = lu_decomposition_dag(tiles=4) # 30 tasks mr = mapreduce_dag(mappers=8, reducers=4) # Random layered DAG with custom structure rand_dag = random_layered_dag( min_depth=6, max_depth=10, min_width=3, max_width=8, ) # Schedule any generated DAG schedule = HeftScheduler().schedule(fog_network(), task_graph) print(f"Makespan: {schedule.makespan:.2f}")
# List all workflows $ dagbench list # Filter by domain $ dagbench list --domain edge-computing # Show detailed info for a workflow $ dagbench info edge.splitstream_pipeline # Validate all workflows $ dagbench validate --all # Show statistics across all workflows $ dagbench stats

Provenance Policy

Every workflow in DAGBench includes honest provenance metadata describing exactly how it was created. We believe transparency about extraction methods is essential for reproducible research.

13
manual-figure
Faithfully extracted from published paper figures or open-source code repositories. Task structures, costs, and edge weights come directly from the original source.
23
programmatic
Generated from code repositories or algorithmic generators (FFT, Cholesky, MapReduce, Gaussian elimination). Structure and costs follow well-known patterns.
14
generated
Synthetic benchmark patterns (chains, diamonds, fork-joins, stencils, trees) and random layered DAGs with configurable structure, designed to stress-test specific scheduler behaviors.
54
ai-generated
AI-generated workflows with honest attribution. Inspired by real applications and papers, but the specific DAG structure was created by an LLM.

Data Format

Each workflow directory contains three files: metadata.yaml (provenance and description), graph.json (SAGA-compatible task graph), and docs.html (visual documentation).

graph.json

{ "header": { "name": "Diamond DAG" }, "graph": { "nodes": [ {"name": "Source", "cost": [10, 12, 14]}, {"name": "TaskA", "cost": [25, 20, 30]}, {"name": "TaskB", "cost": [30, 28, 35]}, {"name": "Sink", "cost": [5, 6, 7]} ], "edges": [ {"source": "Source", "target": "TaskA", "weight": 15}, {"source": "Source", "target": "TaskB", "weight": 12}, {"source": "TaskA", "target": "Sink", "weight": 8}, {"source": "TaskB", "target": "Sink", "weight": 10} ] } }
